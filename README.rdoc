This project is a basic implementation of a user authentication in ruby on rails. 
Once signed up, a user can create articles and modify them.
- Data model

User --------------* Artticle

- Ready to get this up and running ? launch the following ;)

$ git clone https://github.com/slrbl/RailsSignupFromScratch.git ./RailsSignupFromScratch

$ cd RailsSignupFromScratch

$ bundle install

$ rails s
